# Q-GPT Web

ระบบแชตแนว GPT ที่ให้ Q-System ทำงานเบื้องหลัง แล้วตอบแบบสั้น ชัด และอ่านง่าย

## มีอะไรในโปรเจกต์
- หน้าแชตแนว GPT
- Q-System แบบซ่อนเบื้องหลัง
- แผง Admin ดูคะแนนภายใน
- ตั้งค่า API URL / API Key / Model ได้จากหน้าเว็บ
- GitHub Pages workflow สำหรับ deploy อัตโนมัติ

## วิธีใช้งาน
1. เปิด `index.html` ในเบราว์เซอร์ได้ทันที
2. หรือ push โปรเจกต์นี้ขึ้น GitHub
3. เปิด GitHub Pages แล้วเว็บจะ deploy อัตโนมัติ

## ถ้าจะต่อ API จริง
ในหน้าเว็บกดปุ่ม `ตั้งค่า API เพิ่มเติม`
แล้วกรอก:
- API URL
- API Key
- Model

จากนั้นเปิดสวิตช์ `ใช้ API จริงถ้าตั้งค่าไว้`

## หมายเหตุ
เวอร์ชันนี้ทำงานได้เองแบบ local demo
และสามารถต่อ API แบบ OpenAI-compatible ได้
